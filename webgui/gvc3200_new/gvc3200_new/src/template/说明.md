#### template 用于配置项的权限维护，搜索，p值索引等功能

最小级别的配置项结构
```js
{
  label,        // label 标记 方便查看
  p,            // 配置项对应的p值， 一个配置项对应多个p值时，为数组。 非p值配置项不显示
  _p,           // 配置项对应的-p值，当p值不存在时，可以设置 _p值来代替 占位
  lang,         // 配置项对应的翻译词条
  denyOem,      // 特定oem下隐藏
  denyModel,    // 特定设备型号隐藏
  denyRole,     // 特定的登录身份下隐藏（user 或 admin）
  noInit,        // 初始化时不渲染的配置(有p值)
  reboot,        // 是否需要重启
}
```
